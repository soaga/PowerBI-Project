# Data Professionals Survey Analysis | Power BI

## Project Overview
This project analyzes survey responses from data professionals to explore salary trends, job satisfaction, work-life balance, and tool preferences.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Objectives
- Visualize salary distribution across job roles
- Analyze job satisfaction levels
- Identify popular programming languages
- Explore work-life balance trends

## Key Insights
- Python is the most preferred tool among data professionals
- Salary varies significantly by role and experience level
- Work-life balance impacts job satisfaction strongly

## Dashboard Features
- Interactive filters
- KPI cards
- Bar charts and trend analysis
